## Backend Tech
- Mongodb
- Express
- Cors
- Bcrypt
- REST API
- Socket.io

## Features
- Toys CRUD
- Filter
- Sort
- Search
- Dashboard with Charts
- Map with location of stores
- Login, signup and logout
- Chat rooms by topic
